   - Dynamic import: const RunwayScene = dynamic(() => import('@/components/runway/RunwayScene'), { ssr: false })
   - 2D overlay positioned absolute over Canvas:
     * Logo: "TOP 10 PROM" gold, centered top, fade in 1s
     * Tagline: "Your Moment. Your Dress. Your Night." ivory, fade in 1.5s
     * Event selector: Prom / Wedding pill buttons, glass-light, fade in 2s
     * CTA: "Enter the Showroom" gold button, spring animation, fade in 2.5s

     * Bottom: "5+ Boutique Locations Nationwide" platinum, fade in 3s
   - On CTA click: set sessionStorage 'has_entered' + navigate to /(main)

4. Build 3D scene components:
   - components/runway/RunwayScene.tsx:
     * Canvas: fov 45, camera [0, 1.6, 5], dpr [1, 2]
     * <Environment preset="studio" />
     * Key light: directional, #FFF5E6, intensity 1.2, position [5, 5, 5]
     * Rim lights: 2x point, #E8E0FF, intensity 0.6, behind-left + behind-right
     * Ambient: intensity 0.15
     * Fog: linear, #050505, near 15, far 30
     * Three <DressStation> at z=-2, z=-7, z=-12
     * <CameraRig />
     * Post-processing: Bloom (0.3, threshold 0.9), Vignette (0.3, 0.7), subtle film grain
   
   - components/runway/RunwayPlatform.tsx:
     * Plane: 3 x 20, MeshPhysicalMaterial, metalness 0.9, roughness 0.1, color #1A1A1A
     * Gold rails: BoxGeometry 0.02 x 0.02 x 20, emissive #D4AF37, emissiveIntensity 0.5
   
   - components/runway/DressStation.tsx:
     * Props: position, modelUrl, spotlightColor
     * useGLTF for model loading
     * useFrame for y-rotation (0.003 rad/frame)
     * SpotLight: intensity 2, angle 0.3, penumbra 0.5
   
   - components/runway/CameraRig.tsx:
     * useFrame: dolly from z=5 to z=-5 over 12s (lerp)
     * Subtle sine vertical bob: amplitude 0.02, freq 0.5Hz
     * After 12s: enable OrbitControls (constrain azimuth ±45°, polar 60-90°, no zoom)
     * useReducedMotion: skip animation, start at z=-2

5. Build fallback:
